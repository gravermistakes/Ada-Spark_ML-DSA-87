Standing invariants

never violated
  • ⚜Fail-closed is sacred: no gate returns accept until it genuinely verifies.
  • ⚜No frozen or self-derived test vectors — only published standards, pinned reference files, or relational properties.
  • ⚜Done means: builds, every test main prints [PASS] and exits 0, and gnatprove reports 0 unproved.
  • ⚜Constant-time verification; zeroize on failure.

Language policy — non-negotiable. Ada and Bash only, everywhere: library code, tests, tooling, and anything run to produce a committed artifact. No Python (including hashlib), Go, Rust, Ruby, Perl, or JavaScript. If a task seems to need another language, it is done in Ada or Bash, or it stops.
