# Deoxys

This subdirectory is part of the `lthing-spark` Ada/SPARK project.

## Language policy

All source code in this project must be written in **Ada** or **Bash**.
No other scripting or programming languages (Python, Ruby, Perl, etc.)
are permitted. Tooling scripts use Bash; library code, test harnesses,
and cryptographic implementations use Ada (with SPARK annotations where
proof obligations apply).
